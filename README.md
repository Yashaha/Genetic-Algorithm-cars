## Genetic-Algorithm-cars

### 简介
能够自动进化的小车，依据遗传算法。有利于新初学者对遗传算法的理解。

### 版本
- **v 0.3**
  - 增加适应度评价
  - 静止车子自动销毁
  - 染色体转换成二进制编码、解码
  - 可以进行自然选择

- **v 0.2.1**
  - 确定大致框架

- **v 0.2**
  - 确定染色体
  - 增加车子生成数量
  - 增加车子约束条件
  - 修复偶尔不能运行动画的Bug

- **v 0.1**
  - 随机生成逐渐陡峭的地面
  - 随机生成单个车身、轮子
